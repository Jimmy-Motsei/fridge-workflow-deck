Presentation Navigation Test

This page tests the navigation functionality of the presentation deck.

Test Instructions:

  1. Open the main presentation: index.html
  2. Open browser developer tools (F12)
  3. Go to the Console tab
  4. Try the following navigation methods:
    • Click anywhere on the slide
    • Press Right Arrow (→)
    • Press Left Arrow (←)
    • Press Spacebar
    • Press F for fullscreen

Expected Behavior:

  • ✓ Console should show debug messages for each action
  • ✓ Slides should advance/retreat smoothly
  • ✓ Progress bar should update
  • ✓ Slide counter should update
  • ✓ Navigation should work from any slide

Common Issues & Solutions:

  • • If clicks don't work: Check if you're clicking on navigation elements
  • • If keyboard doesn't work: Click on the slide area first to ensure focus
  • • If console shows errors: Check browser console for JavaScript errors
